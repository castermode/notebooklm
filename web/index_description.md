
编写 index.html

实现一个页面，功能如下：

1. 页面主题是：google notebook 代理
2. 有一个“选择文件”的按钮，点击一次可以上传一个文件。可以通过多次点击，上传多个文件
3. 有一个“选择下载路径”的按钮，通过点击，可以选择下载路径
4. 有一个复选框，表示生成文件的格式，类型有：Audio, Video, MindMap, Reports, Flashcards, Quiz, Infographic, SlideDeck
5. 有一个“提交”按钮，点击之后，可以显示上传文件的名字 ， 生成文件的格式 ， 下载路径。另外显示 “生成中 ...”
6. 页面使用 html/js/css 美观简洁
